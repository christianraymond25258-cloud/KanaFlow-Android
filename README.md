# JIKana 🌊

A Japanese character learning app built with Kotlin & Jetpack Compose.

## Features
- Hiragana & Katakana practice with romaji input
- Kanji SRS (Spaced Repetition) multiple choice
- Firebase Authentication & Firestore progress tracking
- Daily streak system
- Beautiful black & sky blue dark theme
- Haptic feedback & smooth animations

## Tech Stack
- Kotlin + Jetpack Compose
- Firebase Auth + Firestore
- MVVM Architecture
- Navigation Compose
